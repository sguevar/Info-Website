4. Random Access Memory (RAM)

Random Access Memory is a crucial part of a computer so that it runs fast and smooth. In simpler terms, RAM is the short-term memory of a computer rather than the long-term memory such as a Hard Drive (HD) or a Solid State Drive (SSD). Basically a computer loads different applications and stores them in its short-term memory so that when you use that application, you can load into it much faster. AN example would be the information you store when you are browsing the internet, different tabs would be stored into the RAM so that you can quickly go back without waiting for it to load again. If your computer starts to slow down significantly when you have many tabs open, it is most likely your RAM slowing it down. To learn more about RAM, Click Here!
