Programming Software

Programming Software is a application or software that people use to create other applications. Basically programmers are the only ones who are using this type of software. A normal user that has no knowledge of coding would not find any use in these programs. These programs allow code to be written, tested, and debugged. There are many different types of programs that allows a user to more easily edit code and have features that allow programmers to create their products.

Some programs are just simply a text editor, a program that allows programmers to write and modify code/text. These text-editors are used mainly by web developers and more experienced programmers. Some examples include: Atom, Visual Studio Code (what I'm currently using), and Notepad ++. IDE's can also be used as simply a text editor but mainly programmers use it to run, debug and test their code. IDE's allow programmers to have many plug-ins to help them create their applications. Examples include: Eclipse, Pycharm, IntelliJ, and Xcode.
